## Getting Started

### Prerequisites

1. npm
  ```bash
  npm i -g npm@latest
  ```
2. MetaMask
  - https://metamask.io/
3. Ganache
  - https://trufflesuite.com/ganache/
4. API Server
  - https://github.com/sleepy-wood/server-api

### Installation

1. Clone the repo
  ```bash
  git clone https://github.com/sleepy-wood/client-web.git
  ```
2. Install NPM packages
  ```bash
  npm i
  ```
3. Run App
  ```bash
  npm run start
  ```
